# self-hosted-app-directory

Search **120+ real self-hostable apps** offline — by tag or free-text. No deps, no account, no API.

## Run
```
python3 sh.py search media        # tag or keyword
python3 sh.py tags                 # all 21 tags
python3 sh.py random               # surprise me
```

## What's inside
Apps across 21 tags: dns, vpn, media, dev, ai, monitoring, photos, productivity,
comms, security, home, infra, knowledge, reading, finance, web, automation, search,
analytics, logging, bookmarks.

## Why
Picking what to self-host next is half the battle. This gives you a fast, offline
lookup of what exists and where to start — no scrolling awesome-selfhosted for 20 min.
